# diagonal matrix(대각 행렬)
```math
 A = \begin{pmatrix}3 & 0 & 0 & 0 \\
0 & 4 & 0 & 0 \\
0 & 0 & 2 & 0 \\
0 & 0 & 0 & 4 \\ \end{pmatrix}
```
